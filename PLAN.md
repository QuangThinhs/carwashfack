# KẾ HOẠCH LÀM VIỆC NHÓM - DỰ ÁN CARWASH

## 1. Các hạng mục công việc chính
* **Quản trị & Thiết kế hệ thống:**
  * Khởi tạo dự án, thiết lập cấu trúc thư mục base và cấu hình Git.
  * Phân tích luồng nghiệp vụ và thiết kế cơ sở dữ liệu (Database Design & SQL Script).
* **Phát triển Frontend (Giao diện):**
  * Xây dựng giao diện các trang chức năng (Trang chủ, Đăng nhập, Đăng ký).
  * Phát triển giao diện các trang quản lý và đặt lịch dịch vụ bằng JSP, HTML/CSS, Bootstrap.
* **Phát triển Backend (Java Web Application):**
  * Xây dựng cấu trúc mã nguồn theo mô hình MVC (Model, DAO, Controller).
  * Cấu hình kết nối cơ sở dữ liệu (DBContext).
  * Viết các Java Servlet xử lý logic xác thực và các chức năng CRUD của hệ thống.
* **Kiểm thử & Hoàn thiện:**
  * Tích hợp giao diện Frontend vào hệ thống Backend.
  * Tiến hành kiểm thử, rà soát và sửa lỗi (Fix bug).
  * Tổng hợp tài liệu và viết báo cáo dự án.

## 2. Tiến độ thực hiện dự kiến (Timeline)
* **Tuần 1:** Khảo sát yêu cầu, thiết kế Database và cấu hình môi trường Git base.
* **Tuần 2:** Xây dựng giao diện tĩnh (HTML/CSS) và triển khai các class kết nối database (DBContext, Model).
* **Tuần 3:** Phát triển các tính năng cốt lõi (Xác thực người dùng, Luồng xử lý đặt lịch chính).
* **Tuần 4:** Ráp giao diện vào Servlet/JSP, hoàn thiện toàn bộ các chức năng nghiệp vụ còn lại.
* **Tuần 5:** Kiểm thử tích hợp toàn bộ hệ thống, sửa lỗi và đóng gói tài liệu báo cáo.
