# KẾ HOẠCH LÀM VIỆC NHÓM - DỰ ÁN CARWASH (2 Thành viên)

## 1. Phân công nhiệm vụ
* **Thịnh (Nhóm trưởng):** Khởi tạo dự án, setup Git, thiết kế Database và code Backend (Java Servlet, Model, DAO).
* **Tiến:** Code giao diện Frontend (JSP, HTML/CSS, Bootstrap), ráp giao diện với Backend và viết báo cáo tổng kết.

## 2. Tiến độ dự kiến (Timeline)
* **Tuần 1:** Chốt luồng tính năng, thiết kế Database, setup Git.
* **Tuần 2:** Tiến cắt HTML/CSS giao diện cơ bản; Thịnh code Base Backend.
* **Tuần 3:** Thực hiện các chức năng chính (Đăng nhập, Quản lý đặt lịch).
* **Tuần 4:** Ghép Frontend và Backend với nhau, cùng nhau test lỗi.
* **Tuần 5:** Fix bug cuối, hoàn thiện source code và báo cáo.
